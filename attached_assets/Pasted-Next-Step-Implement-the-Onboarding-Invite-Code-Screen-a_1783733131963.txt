   - Method: `POST`
   - Payload: Expect JSON containing `{ join_code: string }`.
   - Logic:

     - Search the `tables` database table for a match with the provided `join_code`.

     - If the table is found, check if the user is already a member in `table_members` to prevent duplicates.
     - If not already a member, execute a database insert into `table_members`:
       - `user_id`: current user's UUID
       - `table_id`: the found table's ID
       - `role`: 'member'
       - `credits`: 50 (Seed them with their initial isolated 50 Study Credits for this specific clan).
     - Return a `200 OK` success response with the `table_id`.
